# 📌 Controle de Equipamentos

Aplicação desenvolvida em **Flutter** para cadastro e gerenciamento de equipamentos por setores da empresa.  
O backend foi desenvolvido em **PHP** e o banco de dados é **MySQL** (XAMPP).
